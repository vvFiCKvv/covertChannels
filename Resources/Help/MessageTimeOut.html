Message TimeOut Help


it is the time between two continues packets that represent a reset of the comunication. For example if the Message TimeOut time is 560ms and the Timming Error is 80ms then, we rerpresent a comunication reset for each two continues packages that the space time is more than 480ms. For more error resistance the Message Timeout must be at least great than the sum of One Bit Delay and double of Timming Error.
Message Timeout > One Bit Delay + 2 * Timming Error.
